# Projects

## Geovision: Image Geolocation with ViTs
- **Technologies**: Python, PyTorch
- **GitHub**: github.com/vik-srinivasan/GeoGuessrCV
- Fine-tuned ViT-B/16 and ResNet-50 on a 104-class country-level geolocation task using GeoGuessr images, achieving 88.3% Top-5 accuracy, surpassing human baselines by 3-5x and outperforming GPT-4o.
- Developed an end-to-end geolocation pipeline with Hugging Face APIs, performing error analysis, class-balancing, and hyperparameter tuning to optimize model performance.

## Context-Aware Voice Dictation for Accessibility
- **Technologies**: Python
- **GitHub**: github.com/vik-srinivasan/Audio-Feedback
- Developed a Voice Dictation Tool leveraging Named Entity Recognition (NER), real-time error correction, and a lightweight memory mechanism to enhance transcription accuracy for blind and low-vision users.
- Built backend functionality in Python to integrate speech-to-text (OpenAI Whisper) and NER (using Google Gemini) for proper noun detection and correction.
- Conducted a user study to evaluate the system's effectiveness, gathering feedback on usability and transcription accuracy, and wrote a research paper documenting the findings and implications.

## Data-Driven Archetypes in the Modern NBA
- **Technologies**: Python, R
- **GitHub**: github.com/vik-srinivasan/nba-archetypes
- Designed and implemented a seeded K-Means clustering model to classify NBA players into 10 offensive archetypes using advanced stats, Synergy play type data, and shot distribution metrics, achieving 92% test accuracy.
- Built interactive visualizations and analytical tools to compare archetypes across players, seasons, and team lineups, uncovering trends in player roles and championship team compositions.

## Multilingual Narrative Detection with XLM-R
- **Technologies**: Python, PyTorch
- **GitHub**: github.com/vik-srinivasan/MultiNarDetection
- Developed a multi-task transformer model (XLM-R) to perform concurrent Named Entity Recognition and sentiment analysis for narrative detection in multilingual social media text, focusing on political discourse.
- Engineered a two-phase fine-tuning approach and leveraged Bayesian hyper-parameter optimization, improving model performance across diverse linguistic datasets.
